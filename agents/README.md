# BMPT AI 文档索引

本目录用于存放面向 AI Agent 的专用说明与约束。

## 文档列表

- `agents/ai_quickstart.md`：AI 快速上手。
- `agents/task_recipes.md`：标准任务配方（命令 + 预期信号）。
- `agents/step_contract.md`：`step(models, input)` 严格接口约定。
- `agents/config.schema.json`：训练配置的机器可读 schema。
- `agents/AGENTS.md`：自动化 Agent 行为边界与执行规范。
